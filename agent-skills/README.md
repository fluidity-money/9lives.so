# Agent Skills

Reusable instruction documents for AI coding agents working on the 9lives
codebase. Each file describes a workflow end-to-end so an agent can follow
it without prior project context.

## Available Skills

| Skill | Description |
|-------|-------------|
| [adding-new-markets.md](adding-new-markets.md) | How to create and register new prediction markets (on-chain + GraphQL) |
| [adding-shortterm-markets.md](adding-shortterm-markets.md) | How to add a new asset to the shortterm price prediction UI |
